# platform-mini

> 最轻量的解析`userAgent`,得到必要的平台信息,

## Install

```bash
npm i platform-mini --save
```

## Usage

#### First

```bash
import {platform} from "platform-mini";

platform = {
    isAndroid: false,
    isIos: true,
    isIpad: false,
    isIphone: true,
    isMac: false,
    isWeixin: false,
    isWindows: false,
    version: "13.2.3"
}

```
